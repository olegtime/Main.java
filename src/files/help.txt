List of commands:

info: output information about the collection to the standard output stream;
show: output to the standard output stream all the elements of the collection in a string representation;
add {element}: add a new item to the collection;
update id {element}: update the value of a collection element whose id is equal to the specified one;
remove_by_id id: remove an item from the collection by its id;
clear: clear the collection;
save: save the collection to a file;
execute_script file_name: read and execute the script from the specified file;
exit: terminate the program (without saving to a file);
add_if_min {element}: add a new element to the collection if its value is less than that of the smallest element of this collection;
remove_greater {element}: remove all elements from the collection that exceed the specified;
history: output the last 11 commands (without their arguments);
remove_all_by_status status: remove from the collection all items whose status field value is equivalent to the specified;
average_of_salary: output the average value of the salary field for all items in the collection;
min_by_organization: output any object from the collection whose organization field value is minimal;
